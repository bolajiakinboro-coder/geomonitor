# VorianCorelli GeoMonitor

Satellite monitoring dashboard for:
- Christ Embassy Asese Campus (Ogun State)
- AgriFI Agricultural Land (Southwest Nigeria — 15,000 ha)

Powered by GeoAI + Sentinel-2 via Microsoft Planetary Computer.

## Deploy to Streamlit Community Cloud

1. Upload this folder to a GitHub repository (public)
2. Go to share.streamlit.io
3. Click "New app"
4. Connect your GitHub repo
5. Set main file: `app.py`
6. Click Deploy

## Data Sources
- Sentinel-2 L2A — Microsoft Planetary Computer (free, no API key required)
- 10m resolution, 5-day refresh cycle
- Coverage: All of Nigeria
